A boilerplate for building modern Websites

Boilerplate for building Websites

Features:

  • Responsive layout
  • Auto deployment of content to production server
  • Supports Sass
  • Transpiles ES6 to ES5 via Rollup/Babel
  • Bundles CSS and JavaScript resources
  • Precompiles Handlebars templates

Learn more: https://github.com/smohadjer/boilerplate
